#### GDP + Geography Radar

This page compares the selected six countries using multiple indicators.
The selected countries come from the top 3 and bottom 3 PM2.5 ranking results.

---

#### Indicators

The radar chart includes:
1. Average PM2.5
2. GDP in 2024
3. Average latitude
4. Average longitude
5. Number of PM2.5 records
6. Number of pollutant types

---

#### How to Read This Page

All indicators are rescaled from 0 to 1 so that different units can be compared in one radar chart.
A larger radar shape means the selected country has higher normalized values across more indicators.
